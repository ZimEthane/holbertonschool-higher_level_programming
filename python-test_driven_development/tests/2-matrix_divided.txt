2-matrix_divided
================

Import the function:

>>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Basic division:

>>> matrix = [[1, 2, 3], [4, 5, 6]]
>>> matrix_divided(matrix, 3)
[[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]
>>> matrix
[[1, 2, 3], [4, 5, 6]]

Works with floats and rounds to 2 decimals:

>>> matrix_divided([[1.5, 2.5]], 2)
[[0.75, 1.25]]
